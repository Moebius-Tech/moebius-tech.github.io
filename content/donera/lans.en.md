# Länsförsäkringar

1. Open the Länsförsäkringar app or online banking and go to transfer
2. Select the account you want to transfer from, then select recipient and add a new recipient with account-clearing number 84202 8143866641
3. Select amount, and then select "Recurring transfer" if you want to make a recurring donation. If so, select monthly.
4. Check that everything is correct, and then approve the payment. Thank you for your contribution! :))
