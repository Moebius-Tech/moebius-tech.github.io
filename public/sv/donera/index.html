Donera till Möbius

Du kan donera pengar till Möbius på flera olika sätt genom olika banker. Välj din bank nedan för att komma till instruktionerna:

  • Donera via Swedbank
  • Donera via Danske Bank
  • Donera via Nordea
  • Donera via Länsförsäkringar
  • Donera via Handelsbanken
  • Donera via annan bank
Om du har valt att donera åtminstone 50kr per månad så är du behörig för att få Möbius tidning (när den väl trycks)! Fyll i här så kan vi nå dig när nästa exemplar trycks.
